# types

## install

```bash title="pnpm"
pnpm add -D @cmtlyt/lingshu-toolkit
```

```bash title="shadcn"
npx shadcn@latest add https://cmtlyt.github.io/lingshu-toolkit/r/sharedTypes.json
```

## usage

```ts
import { types } from '@cmtlyt/lingshu-toolkit/shared'
// or
import { types } from '@cmtlyt/lingshu-toolkit/shared/types'
```
