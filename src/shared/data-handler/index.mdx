# dataHandler

> package version >0.0.0
>
> shadcn any version

## install

```bash title="npm"
npm i @cmtlyt/lingshu-toolkit
```

```bash title="shadcn"
npx shadcn@latest add https://cmtlyt.github.io/lingshu-toolkit/r/sharedDataHandler.json
```

## usage

```ts
import { dataHandler } from '@cmtlyt/lingshu-toolkit/shared'
// or
import { dataHandler } from '@cmtlyt/lingshu-toolkit/shared/data-handler'
```
