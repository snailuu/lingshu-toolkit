# useMount

> package version >0.1.1
>
> shadcn any version

## install

```bash title="pnpm"
pnpm add -D @cmtlyt/lingshu-toolkit
```

```bash title="shadcn"
npx shadcn@latest add https://cmtlyt.github.io/lingshu-toolkit/r/reactUseMount.json
```

## usage

```ts
import { useMount } from '@cmtlyt/lingshu-toolkit/react'
// or
import { useMount } from '@cmtlyt/lingshu-toolkit/react/use-mount'
```
